# Промо-материалы

Публичные слайды и материалы для докладов, демо и онбординга — не входят в публикуемые SDK-пакеты.

|----------|----|----|

## Открыть локально

```bash
open promo/presentation/index.html      # English
open promo/presentation/index.ru.html   # Русский
```

Все стили — в [`presentation/assets/`](presentation/assets/) (`tokens.css`, темы, `deck.css`). Клавиши: ← → Space, Home/End; свайп на мобильных.
