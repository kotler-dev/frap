Self-Healing: как находим элемент

Когда primary селектор сломался — ищем по сигнатуре

1
Primary selector не найден
[data-testid="submit-btn"] → null
2
Извлекаем сигнатуру оригинала
tag:button, role:submit, depth:5, children:1
3
Сканируем все кандидаты
Levenshtein + структурная схожесть → score для каждого
4
Confidence ≥ threshold → HEALED
Новый элемент найден, отчёт с diff сохранён
